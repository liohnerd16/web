# GUIDE TO EDITING CODE & MANAGING CATEGORIES / HƯỚNG DẪN SỬA CODE & QUẢN LÝ DANH MỤC

Primary file for editing: `server/database/db.js`
File chỉnh sửa chính: `server/database/db.js`

---

## 1. Data Management (Turso Cloud)
Your database is now on the cloud (**Turso**). 
Cơ sở dữ liệu hiện tại được lưu trên đám mây (**Turso**).

- **Local Edit**: Changes in `server/database/db.js` (samples array) will sync to the cloud when you deploy.
- **Production Update**: Run `vercel --prod` to apply code and database changes to the live site.

---

## 2. Managing Categories
Categories are generated **automatically** from your video entries. 

### To Add or Rename:
1. Update the category name at position #4 in the `samples` array in `db.js`.
2. Deploy: `vercel --prod`.

---

## 3. Multilingual Text (I18N)
UI text (buttons, menus, toasts) is managed in: `public/script.js`.

1. Find the `const I18N = { ... }` variable.
2. Edit English (`en`) or Vietnamese (`vi`) keys.
3. To auto-sync translations: `npm run sync`.

---

## 4. Dashboard & Project Editing
The dashboard includes advanced tools for content management:

- **Bilingual Edit**: The edit modal allows updating both EN and VI versions of titles, descriptions, and topics simultaneously.
- **Live LaTeX Preview**: Click the "Preview" button in the edit modal to see a live rendering of the scientific explanation (including MathJax and diagrams) before saving.
- **Localized Table**: The project management table is fully bilingual and switches based on the global language toggle.

---

## 5. Maintenance Workflow (IMPORTANT)
Before pushing any major code changes:
1. **Sync Manifest**: `npm run sync-manifest` (Generates "What's New" news entries).
2. **Check Images**: `npm run check-images` (Verifies LaTeX diagrams).
3. **Deploy**: `vercel --prod`.

---

## 5. Security (Secrets)
Never hardcode API Keys. Manage them via:
- **Local**: `.env` file.
- **Production**: Vercel Dashboard or `vercel env add`.
